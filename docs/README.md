# Documentación orientada a agentes de IA — xyflow

Esta carpeta está diseñada para que un agente (o equipo técnico) pueda entender rápido el monorepo y ejecutar cambios con bajo riesgo.

## Orden recomendado de lectura

1. [`playbook-para-agentes-ia.md`](./playbook-para-agentes-ia.md)
2. [`arquitectura-monorepo.md`](./arquitectura-monorepo.md)
3. [`superficie-publica-y-extensiones.md`](./superficie-publica-y-extensiones.md)
4. [`workflows-de-desarrollo-y-validacion.md`](./workflows-de-desarrollo-y-validacion.md)
5. [`catalogo-de-ejemplos.md`](./catalogo-de-ejemplos.md)
6. [`troubleshooting.md`](./troubleshooting.md)

## Objetivo

- Reducir tiempo de onboarding técnico.
- Estandarizar cómo un agente analiza, implementa y valida cambios.
- Alinear el trabajo sobre `@xyflow/react`, `@xyflow/svelte` y `@xyflow/system`.
